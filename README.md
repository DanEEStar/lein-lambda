# aws-lambda

A Leiningen plugin to automate AWS Lambda deployments

## Usage

A work in progress - use at your own risk.

Put `[aws-lambda "0.1.0-SNAPSHOT"]` into the `:plugins` vector of your project.clj.

    $ lein lambda deploy
